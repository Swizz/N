# N
Notification when a cli command finishes

## Installation
`curl https://raw.githubusercontent.com/madhavanmalolan/N/master/install.sh | sudo sh -`

## Usage 
`n <bash command>`

For example, 
`n cp very_large_file.bin /tmp/`
